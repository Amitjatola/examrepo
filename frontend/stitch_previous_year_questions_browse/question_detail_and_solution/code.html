Question 12

Algebra
If x > 0 and x2 - 5x + 6 = 0, what is the value of x?

lightbulb Solution & Explanation

ID: #QN-8921
check_circle

The quadratic equation factors into (x - 2)(x - 3) = 0, meaning x can be either 2 or 3. Since both values are greater than 0, Option C is correct.

Step-by-Step Breakdown

1

Write down the given equation.

x2 - 5x + 6 = 0
2

Find two numbers that multiply to +6 and add to -5.

The numbers are -2 and -3.

3

Factor the equation.

(x - 2)(x - 3) = 0

Therefore, x = 2 or x = 3. Both satisfy the condition x > 0.
